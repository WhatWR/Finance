Ch8 — Cost of Capital (WACC)

Cost of Debt, Cost of Equity (CAPM/DDM), Cost of Preferred, WACC, Floatation

ข้อ 1 — บริษัท Leo: WACC ครบทุกขั้นตอน

ราคาหุ้น 35, จำนวนหุ้น 150 ล้าน, IB Debt 3,400 ล้าน, Total Liab 4,600, Equity 3,300
β=1.55, t=20%, Rf=3.45%, MRP=6.35%, Yield Spread=2.5%
1) Market Capitalization (E)
E = Price × Shares = 35 × 150 = 5,250 ล้าน
2) Interest-Bearing Debt (L)
L = 3,400 ล้าน (ไม่ใช้ Total Liab)
3) Firm Value
A = L + E = 3,400 + 5,250 = 8,650 ล้าน
4) Cost of Debt (Rd)
Rd = Rf + Spread = 3.45% + 2.5% = 5.95%
5) After-tax Cost of Debt
Rd(1−t) = 0.0595 × 0.80 = 4.76%
6) Cost of Equity (CAPM)
Re = Rf + β·MRP
= 0.0345 + 1.55·0.0635 = 13.29%
7) Wd, We
Wd = 3,400 / 8,650 = 0.3931
We = 5,250 / 8,650 = 0.6069
8) WACC
WACC = Wd·Rd(1−t) + We·Re
= 0.3931·0.0476 + 0.6069·0.13293
= 0.01871 + 0.08068
= 9.94%
WACC = 9.94% ต่อปี

ข้อ 2 — บริษัทสิงห์: Cost of New Equity + Floatation

ราคา underwritten = 40 บาท | f = 1% | D0 = 3.2 (XD แล้ว) | ROE = 14% | Payout = 65%
1) g
RR = 1 − 0.65 = 0.35
g = ROE × RR = 0.14 × 0.35 = 4.9%
2) D1
D1 = D0(1+g) = 3.2 × 1.049 = 3.3568
3) Cost of Equity (XD แล้ว → ใช้ Pxd0(1−f) เป็นเงินสุทธิที่ได้)
Re = D1 / [Pxd0·(1−f)] + g
= 3.3568 / [40 × 0.99] + 0.049
= 3.3568 / 39.60 + 0.049
= 0.08477 + 0.049 = 13.38%
ทำไมต้องคิด Floatation? เพราะบริษัทไม่ได้รับเต็ม 40 บาท — เสีย 1% เป็นค่าธรรมเนียม → เงินจริงที่ได้คือ 39.60 บาทต่อหุ้น แต่ต้องจ่ายปันผลตามมูลค่าจริง ผลทำให้ "ต้นทุนแพงขึ้น"
Re = 13.38% ต่อปี

ข้อ 3 — บริษัทช้างเผือก: เมื่อไรต้องออกหุ้นเพิ่มทุน

Wd=40%, We=60% (จากกำไรสะสม) | Payout=55% | NI=850 ล้าน
หาขนาด Capex ที่กำไรสะสมยังพอ
Retained Earnings = NI × (1 − Payout) = 850 × 0.45 = 382.5 ล้าน

We = 0.60 → ต้องการ equity 60% ของ Capex
Threshold Capex = 382.5 / 0.60 = 637.5 ล้าน
หลักคิด: ตราบใดที่ Capex ≤ 637.5 ล้าน → กำไรสะสมที่เหลือ 382.5 ล้านพอเป็น 60% — ไม่ต้องออกหุ้นใหม่ (และไม่มี floatation cost) แต่ถ้าเกินกว่านั้น ต้องออกหุ้นใหม่ → ต้นทุนสูงขึ้น
บริษัทจะออกหุ้นเพิ่มทุนเมื่อ Capex > 637.5 ล้านบาท

ข้อ 4 — บริษัทสิงหราช (เล็ก ไม่จดทะเบียน → ใช้ Own-Bond Yield)

MLR=6.25%, Spread=1.75%, t=20%, Wd=40%, We=60%, MRP=6.25%
1) Cost of Debt
Rd = MLR + Spread = 6.25% + 1.75% = 8.0%
2) After-tax Cost of Debt
Rd(1−t) = 0.08 × 0.80 = 6.4%
3) Cost of Equity — วิธี 3 (Own-Bond Yield)
Re = Rd + MRP = 0.08 + 0.0625 = 14.25%
ใช้เมื่อ: บริษัทเล็ก, ไม่มีข้อมูล Beta ใช้ CAPM ไม่ได้ — ใช้ Rd ของตัวเองเป็น proxy + bonus 5–6% (MRP) เพราะ equity เสี่ยงกว่าหนี้
4) WACC
WACC = 0.4·0.064 + 0.6·0.1425
= 0.0256 + 0.0855 = 11.11%
WACC = 11.11%

ข้อ 5 — Cost of Equity เมื่อ Beta ติดลบ

β = -0.5 (Rm ขึ้น 1% → R ของหุ้นลด 0.5%)
Rf 1ปี = 1.2%, Rf 10ปี = 3.2%, Rm = 8%
ใช้ Rf 10 ปี เป็นมาตรฐาน (long-term equity → match กับ duration ของกระแสเงินสดบริษัท)
CAPM: Re = Rf + β(Rm − Rf)
= 0.032 + (−0.5)(0.08 − 0.032)
= 0.032 + (−0.5)(0.048)
= 0.032 − 0.024
= 0.008 = 0.8%
เข้าใจ negative beta: หุ้นเคลื่อนสวนตลาด — เป็น hedge ในยามตลาดล่ม → นักลงทุนยอมรับผลตอบแทนต่ำเพราะลดความเสี่ยง portfolio ได้
Re = 0.8% ต่อปี

ข้อ 7 — บริษัท CMMU: Cost of Preferred Stock

ราคา underwritten = 20 บาท | f = 2% | Dividend = 2 บาท/ปี | t = 20%
สูตร (preferred → g = 0)
Rp = D / [P·(1−f)]
= 2 / [20 × 0.98]
= 2 / 19.6
= 10.20%
หมายเหตุ: Preferred dividend จ่ายจาก net profit (after-tax) → ไม่หักภาษีเหมือนหนี้ → ใช้ Rp ตรงๆ
Rp = 10.20%

ข้อ 8 — WACC พื้นฐาน

Wd=0.5, We=0.5, Rd=8.5%, Re=15%, t=40%
WACC = Wd·Rd(1−t) + We·Re
= 0.5·0.085·(1−0.4) + 0.5·0.15
= 0.5·0.051 + 0.075
= 0.0255 + 0.075
= 0.1005 = 10.05%
WACC = 10.05%

ข้อ 9 — แก้สมการกลับหา Re จาก WACC

Wd=0.4, We=0.6, Rd=5%, t=20%, WACC=8.5%
ย้ายข้างสมการ
WACC = Wd·Rd(1−t) + We·Re
0.085 = 0.4·0.05·0.8 + 0.6·Re
0.085 = 0.016 + 0.6·Re
0.069 = 0.6·Re
Re = 0.069 / 0.6 = 0.115 = 11.5%
Re = 11.5%

ข้อ 10 — การเปลี่ยนภาษีกระทบ Cost of Debt อย่างไร

Par=1,000, n=20 ปี, Coupon 7%/ปี (m=2), t1=40%, t2=30% (ลดภาษี)
หา Rd จาก TVM (P/Y=2)
N=40, PV=−1,000, PMT=35 (=1,000·0.07/2), FV=1,000
CPT I/Y = 7% (Before-tax)
เปรียบเทียบ After-tax Rd
Scenario I (t=40%): Rd(1−t) = 0.07 × 0.60 = 4.2%
Scenario II (t=30%): Rd(1−t) = 0.07 × 0.70 = 4.9%

ΔWACC (Δ Cost of Debt) = 4.9% − 4.2% = +0.7%
เข้าใจ Tax Shield: ภาษีลดลง → Tax Shield ของหนี้ลดลง → ต้นทุนหนี้ after-tax สูงขึ้น → WACC สูงขึ้น
ΔWACC = +0.7% (เพิ่มขึ้น)
